## ğŸ†˜ Troubleshooting

### "Text is too small, I can't read it"
**Solution:**
1. Enable Large Text (instant +12%)
2. Move Text Size slider to 110%
3. Enable High Contrast for better visibility
4. Consider using screen reader if still difficult

---

### "Text is overlapping buttons"
**Problem:** Impossible with our safety limits!
**But if it happens:**
1. Click "Reset to Defaults"
2. Use Large Text toggle only (not slider)
3. Report the issue to support

---

### "Animations make me dizzy"
**Solution:**
1. Enable "Reduce Motion" toggle
2. This minimizes all dice rolls and transitions
3. Also recommended: Take breaks every 30 minutes

---

### "Colors look weird/wrong"
**Solution:**
1. Check if Color Blind Mode is accidentally enabled
2. If you are color blind, select the correct type
3. Try High Contrast mode
4. Reset to defaults if colors are broken

---

### "I changed settings but nothing happened"
**Solution:**
1. Make sure you clicked "Save Settings"
2. Reload the page
3. Check if settings actually saved (reopen Settings panel)
4. Clear browser cache if issue persists
